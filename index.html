    Automated Stock Price Tracker Using Web Scraping

    This project involves creating an automated bot to track stock data from Yahoo Finance using Python. The script leverages the requests_html library to scrape real-time stock details such as stock name, price, open price, and previous close price. The extracted data, including the current date and time, is then saved into a Google Spreadsheet using the gspread library.

    Key Features:

    Web Scraping: Extracts stock information from Yahoo Finance dynamically.

    Google Sheets Integration: Saves the stock data into a Google Sheet for easy tracking and analysis.

    Automated Execution: Designed for daily use, automating the data collection process without manual intervention.

    This solution is efficient for tracking stock prices consistently and can be extended to monitor multiple stocks or generate insights over time.

    Amazon Product Scraper Using Scrapy

    This project is a Python-based web crawler built using the Scrapy framework to extract product data from Amazon based on a user-specified keyword. The crawler gathers information such as product name, price, ratings, product description, original price, discount percentage, and image URLs, storing it in a structured format like CSV or JSON.
    Key Features:

    Dynamic User Input: Users can enter search strings like "shirts for men", which dynamically form search URLs to crawl relevant products.

    Data Extraction: Scrapes critical product attributes including: Product URL, Product Name, Price and Original Price, Discount Percentage, Product Ratings, Image URL, Product Description,

    Avoiding Blocks: Integrated with ScrapeOps API to rotate headers and user-agents on each request to prevent detection and blocking. Pagination Support: Automatically navigates through multiple pages of search results, scraping up to hundreds of items.

    Data Cleaning: Utilizes pipelines to clean and format the scraped data, such as removing whitespace and adding appropriate symbols (₹ for prices).
